# SUPPORA - BPO Landing Page

Современный лендинг для BPO компании SUPPORA, созданный с использованием NextJS, TypeScript и TurboPack.

## 🚀 Особенности

- **Современный дизайн** с использованием заданной цветовой схемы
- **Полностью адаптивный** - работает на всех устройствах
- **Интерактивные анимации** и плавные переходы
- **Оптимизированная производительность** с Vite
- **SEO-оптимизированный** с мета-тегами
- **Доступность** - поддержка screen readers и клавиатурной навигации

## 🎨 Цветовая схема

- **Основной цвет**: #855CE4
- **Акцентный цвет**: #7B53F3
- **Фоновый цвет**: #1E1E1E
- **Цвет текста**: #E7E7E7
- **Основной градиент**: #6532FF → #6E51C8
- **Градиент акцента**: #9D4CFF → #6E51C8
- **Градиент фона**: #1E1E1E → #2D2C31

## 📱 Секции лендинга

1. **Header** - Навигация с логотипом и CTA
2. **Hero** - Главный экран с призывом к действию
3. **About** - Информация о компании и ценностях
4. **Services** - Карточки с описанием услуг
5. **Advantages** - Преимущества и статистика
6. **Testimonials** - Отзывы клиентов с каруселью
7. **Contact** - Форма обратной связи и контакты
8. **Footer** - Дополнительные ссылки и информация

## 🛠 Технологии

- **NextJS 15 + React 19** - UI библиотека
- **TypeScript** - Типизация
- **TurboPack** - Сборщик и dev сервер
- **SASS** - Стилизация с SASS/CSS переменными
- **ESLint** - Линтинг кода

## 🚀 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```
